#CALCULADORA DE CONVERSIÓN
Este proyecto es un sistema en Java permite convertir números entre distintos sistemas de numeración, como decimal, binario, octal y hexadecimal.
Su objetivo es facilitar el aprendizaje y la comprensión de las conversiones entre bases numéricas de forma rápida y precisa.

Sus funcionalidades son:
1. Convertir decimal a binario
2. Convertir decimal a hexadecimal
3. Convertir binario a decimal
4. Convertir hexadecimal a decimal
5. Mostrar historial de conversiones
6. Salir

Su entorno es IntelliJ Idea en una versión de JDK 25
Cómo ejecutar el proyecto
1. Clona el repositorio:
git clone https://github.com/karlaabet/C-DIGO-GENERAL-CALCULADORA.git
2. Abre el proyecto en IntelliJ IDEA (o cualquier otro IDE compatible con Java).
3. Compila y ejecuta el archivo principal:
javac Conversiontool.java
java Conversiontool
